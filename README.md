# LiftForge 🏋️

**Track Heavy. Compare Gyms. Build Your Legacy.**

A mobile-first fitness tracking PWA for powerlifters and gym influencers.

## Features
- **Dashboard** — Stats, streaks, motivational quotes, hot gym of the week
- **Workout Logger** — Template-based or freestyle, with weight/reps/RPE, rest timer, PR detection
- **Progress & Analytics** — Volume charts, 1RM curves, PR table, body measurements
- **Gym Comparison Hub** — Rate, review, and compare gyms side-by-side
- **Influencer Toolkit** — Generate branded PR posts with captions for Instagram/TikTok

## Deploy
This is a static site. Push to GitHub, connect to Vercel, deploy.

## Tech
React 18 (CDN) + Babel (in-browser JSX compilation) + localStorage persistence
